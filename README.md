# IoT-DevKit

![Open-source is Love](./arduino.jpg)

IoT开发工具箱：收纳整理IoT开发中可能会用到的、可复用的一些库和配置等。

原有的分散的仓库全部删除，其内容全部转移到本仓库。由于这些工具提交后几乎从未修改，可以认为是处于归档状态，因此不保留旧仓库的提交修改记录，仅记录首次提交日期。

2019年9月15日整理。

**<span style="color:red;">注意：提交前必须对敏感信息脱敏。</span>**
